# Daily Reports Service

Responsibilities:

- Daily report listing
- Daily report creation
- Daily report update
- Daily report deletion

Target routes:

- `GET /api/daily-reports`
- `POST /api/daily-reports`
- `PUT /api/daily-reports/:reportId`
- `DELETE /api/daily-reports/:reportId`
